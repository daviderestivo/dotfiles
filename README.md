# dotfiles

This repo collects my personal dofiles:
- .bashrc
- .bash_profile
- .tmux.conf

## Installation

### Clone the repo
```
git clone https://github.com/daviderestivo/dotfiles.git
```

### Link them 

```
ln -s dotfiles/bash_profile .bash_profile
ln -s dotfiles/tmux.conf .tmux.conf
ln -s dotfiles/bashrc .bashrc
```
